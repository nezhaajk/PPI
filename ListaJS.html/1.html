O que é JavaScript?

A motivaçâo para a criação do JavaScript foi a necessidade de fazer com que as paginas da internet fossem mais dinamicas e interativas. A execuçâo do JavaScript no navegador é pelo cliente, no servidor é usado em um ambiente de execução especifico,para lidar com a logica do aplicativo e interagir com bancos de dados e sistema de arquivo. Sandbox é um ambiente de teste isolado,seguro e controlado onde podem ser executados e analisados programas, sem correr o risco de prejudicar o sistema principal ou a rede. Objetos globais como document, console e alert são objetos pré-escritos que estão sempre disponiveis em qualquer parte do codigo. Document: é o documento html da pagina atual no navegador. atraves dele, é possivel acessar e manipular elementos da pagina, como adicionar ou remover conteudo. Console permite que você envie mensagens para o console do navegador. Alert é uma forma de interagir com o usuario, apresenta uma caixa de dialogo na pagina para mostrar uma mensagem pro usuario.
